# React Boilerplate

This boilerplate includes a basic React app setup:

#### Libraries

- React 
- React DOM
- Redux
- React Redux
- Redux Thunk

#### Build Tools

- Webpack 2
- Babel (Stage 1 + Regenerator Runtime)
- ESLint

## Installation

```bash
yarn
```

OR

```bash
npm install
```

## Running Dev Server

```bash
npm run dev
```

## Building and Running Production Server

```bash
npm run build
npm run start
```
